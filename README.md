Proyecto desarrollado como parte del periodo de evaluación técnica 
para la vacante de Becario GCP & Workspace en Turing Inteligencia Artificial.

Implementa una infraestructura en Google Cloud Platform que detecta 
automáticamente archivos subidos a un bucket de Cloud Storage, extrae 
sus metadatos mediante una Cloud Function en Python, y los registra 
en Cloud Logging. El Día 2 extiende la solución con automatizaciones 
en Google Workspace usando Apps Script.
